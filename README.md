# 起動
右端のボタンでコピーしたあと、ターミナルに貼り付けると一括で実行できます。
```
# リポジトリをクローンする
git clone https://github.com/itc-s21009/typing_game.git
# クローンしたフォルダに移動する
cd typing_game
# 必要なパッケージをダウンロードする
npm i
# 公開するファイルをビルドする
npm run build
# 起動する
npm run start
```
